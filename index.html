Task 1:

1. CPU (Central Processing Unit) - The brain of the computer that processes instructions and manages data flow.

2. RAM (Random Access Memory) - Temporary, fast storage used to hold data and instructions while programs are running.

3. Motherboard - The main circuit board that connects all components, allowing them to communicate.

4. Power Supply Unit (PSU) - Converts electrical power from an outlet into usable power for computer components.

5. Hard Drive (HDD) - A magnetic storage device used to permanently store files and programs.

6. Solid-State Drive (SSD) - A faster storage device that uses flash memory instead of moving parts.

7. Graphics Card (GPU) - Handles image and video rendering, improving performance in games and visual tasks.

8. Peripherals - External devices like keyboards, mice, and printers that allow interaction with the computer.

9. Input Devices - Hardware used to provide data to the computer (e.g., mouse, keyboard, scanner).

10. Output Devices - Hardware that shows or delivers results from the computer (e.g., monitor, speakers, printer).

Task 3: Internal Components (Diagram + Labeling)

CPU (Inside the cooler)

The CPU (Central Processing Unit) is the brain of the computer, handling instructions and processing data. It determines how fast and efficiently your computer can think and perform tasks.

GPU

The GPU (Graphics Processing Unit) renders images, videos, and animations for display. It is essential for gaming, 3D modeling, and accelerating tasks like video editing or AI processing.

RAM

RAM (Random Access Memory) temporarily stores data so the CPU can access it quickly. More RAM allows you to multitask smoothly without slowing down the system.

SSD

The SSD (Solid State Drive) stores your operating system, applications, and files permanently. It is much faster than a traditional hard drive, making booting and loading times shorter.

Motherboard

The Motherboard connects all the components of the computer so they can communicate with each other. It also supplies power and ensures compatibility between your hardware parts.

Task 4: Compare & Contrast (RAM vs ROM, HDD vs SSD)

RAM vs. ROM:

RAM is temporary memory that stores data only while the computer is running, making it fast and essential for multitasking. ROM is permanent memory that stores firmware or startup instructions; it doesn't lose data when power is off. RAM offers speed and flexibility, while ROM provides stability and essential boot information.

"Any sufficiently advanced technology is indistinguishable from magic" - Arthur C. Clarke

HDD vs. SSD:

HDDs use spinning magnetic disks to store data, offering large capacity at a lower price but slower speeds. SSDs use flash memory with no moving parts, making them faster, more durable, and energy efficient. HDDs are best for bulk storage, while SSDs provide speed and performance.
